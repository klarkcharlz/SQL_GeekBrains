Задание № 2
При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, поиск электронного адреса пользователя по его имени.

Будем использовать тот же вариант с хэш таблицами.

hset user_email Mike 'abc@mail.com' - по имени хранится почта
hset email_user abc@mail.com 'Mike' - по почте хранится имя

hget user_email Mike - найдем почту
hget email_user abc@mail.com - найдем имя
